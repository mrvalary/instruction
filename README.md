## Введение



Git — это распределённая система контроля версий. Она позволяет сохранять историю изменений проекта, работать с ветками и объединять изменения.



GitHub — это онлайн-сервис для хранения Git-репозиториев и совместной работы над проектами.



---



## Создание учётной записи GitHub



Для создания учётной записи GitHub необходимо:



1. Перейти на сайт GitHub.

2. Нажать кнопку Sign up.

3. Ввести email.

4. Придумать пароль.

5. Указать имя пользователя.

6. Подтвердить регистрацию через email.

7. После входа в аккаунт создать новый репозиторий.
